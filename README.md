# 🕌 Muslim Tracker

A Flutter application to help **manage and monitor Muslim participants** in mosques during the last ten nights of Ramadan.

This application provides tools to:

* 📊 Monitor total participants
* ✅ Track attendance (hadir / tidak hadir)
* 📍 Display participant origin statistics
* 📱 Responsive dashboard (Mobile, Tablet, Desktop)

The goal of this project is to **assist mosque committees in organizing Muslim activities efficiently and transparently**.

---

# ✨ Features

* 📊 Dashboard summary (Total Peserta, Hadir, Tidak Hadir)
* 📍 Chart statistik asal peserta
* 📋 Absensi peserta
* 📱 Responsive layout (Flutter Web / Mobile)
* ⚡ Lightweight and easy to deploy

# 🚀 Getting Started

Make sure you have Flutter installed.

https://docs.flutter.dev/get-started/install

Clone the repository:

```
git clone https://github.com/endrose/itikaf-tracker-apps.git
```

Go to project directory:

```
cd Muslim_tracker
```

Install dependencies:

```
flutter pub get
```

Run the project:

```
flutter run
```
